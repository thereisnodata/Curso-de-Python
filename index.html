Análisis y Visualización de Datos con Python

Instructor: Angel Miguel Alvarez Calicho

Prerrequisitos: Estadística Básica

Objetivos

Objetivo General:

Dominar el manejo de Python para el análisis de datos y la visualización utilizando diversas bibliotecas como Pandas, NumPy, Matplotlib, Seaborn y otras herramientas relevantes para llevar a cabo un análisis exhaustivo de conjuntos de datos.

Objetivos Específicos:

  • Familiarización con el entorno de Python para análisis de datos
  • Manejo de Bases de Datos y manipulación de variables
  • Análisis Descriptivo de Datos
  • Visualización de Datos
  • Edición y Presentación de Resultados
  • Exportación de Resultados

Este curso proporcionará a los participantes las habilidades necesarias para realizar un análisis detallado de datos utilizando Python, desde la importación y limpieza de datos hasta la presentación visual y la exportación de los resultados en varios formatos.
